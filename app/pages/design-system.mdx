# Design System

## Typography

For typography two fonts are mainly used: [Space Grotesk](https://floriankarsten.github.io/space-grotesk/) and [Inter](https://rsms.me/inter).
For monospace Menlo is preferred.

For headings and other prominent UI elements Space Grotesk is used.
Normal text and smaller UI elements are set in Inter.


## Colors

The base colors are a set of colors from which the light and dark colors are derived.
Light colors occupy a slice of each colors shades and dark colors do the same.

The accent color is a blueish purple.
In addition there are green and red for positive and negative feedback.

There are often many shades of either very light or very dark greys needed, those are included as well.
